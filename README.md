# Application d'introspection

Petite application web locale pour faire un point rapide sur ton etat du jour:

- humeur
- niveau d'energie
- 3 questions de reflexion
- synthese automatique
- historique local (stocke dans le navigateur)

## Lancer l'application

Option la plus simple:

1. Ouvre le dossier `app-introspection`
2. Double-clique sur `index.html`

Option serveur local (facultatif):

```powershell
cd C:\Users\PC\Documents\app-introspection
python -m http.server 8080
```

Puis ouvre [http://localhost:8080](http://localhost:8080)
